Quick start
===========

Automatically
-------------

Run init script

```
$ ./init.sh
```

Use sudo if script needs privileges

Manually
--------

*   Step 1: Clone repository
*   Step 2: Install dependencies

```
$ npm install
```

*   Step 3: Install NW.js 0.17.6 SDK

```
$ npm i -g nwjs
$ nw install 0.17.6-sdk
$ nw use 0.17.6-sdk
```

*   Step 4: Run the app

```
$ npm start
```
